# 🏥 Hospital Management Dashboard

## 📌 Project Overview

This project presents an interactive Hospital Management Dashboard developed using Power BI. It analyzes hospital data to provide meaningful insights into patient admissions, department performance, doctor efficiency, revenue, and healthcare operations.

## 🎯 Objectives

- Analyze hospital performance
- Monitor patient admissions
- Track department-wise performance
- Evaluate doctor efficiency
- Visualize hospital KPIs
- Support data-driven decision making

## 🛠️ Tools & Technologies

- Power BI
- Microsoft Excel
- Power Query
- DAX

## 📊 Key Performance Indicators (KPIs)

- Total Patients
- Total Doctors
- Total Revenue
- Average Treatment Cost
- Bed Occupancy Rate
- Patient Satisfaction
- Department-wise Patients

## 📈 Key Insights

- Cardiology department has the highest number of patients.
- Emergency department receives the maximum admissions.
- Revenue varies across departments.
- Patient satisfaction remains consistently high.
- Dashboard enables quick monitoring of hospital performance.
